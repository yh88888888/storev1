# Table Setting

## store_tb

```sql
create table store_tb (
    id integer generated by default as identity,
    price integer,
    stock integer,
    name varchar(255),
    primary key (id)
);
```

## log_tb

```sql
create table log_tb (
    id integer generated by default as identity,
    store_id integer
    qty integer
    total_price integer
    buyer varchar(255),
    primary key (id)
);
```
